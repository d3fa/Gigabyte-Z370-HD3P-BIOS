# CoffeeMod bios for Gigabyte Z370 HD3P

### Support QTJ0, QTJ1, QTj2 and other CPU
Just Flash any .bin from BIOS or programmer - Main BIOS (M)

### Whats new?

**Add new UEFI options:**
* Peripherals -> Memory Configuration (Extended) // for better memory overclocking 
* Peripherals -> CPU Lock Options (Extended) // 'CFG Lock' for MacOS
  
**Updates:**
* CPU microcode 906ED to 'FC'
* GOP 9.0.1112

> [!TIP]
> Notice about microcodes
```
506E3 – 6 Gen Skylake;
506E8 – QL2X, QL3X;
906E9 – 7 Gen Kaby Lake;
906EA – 8 and 9 Gen Coffee Lake 6-core, QNCT, QNVH;
906EB – 8 and 9 Gen Coffee Lake 4-core;
906EC – 9 Gen Coffee Lake Refresh, stepping P0, QQLT, QQLS;
906ED – 9 Gen Coffee Lake Refresh, stepping R0, QTJ0, QTJ1, QTJ2, SRFD0 (9980HK).
```
